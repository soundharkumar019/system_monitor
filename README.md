# System Monitor Script

A simple Bash script to monitor basic system health on Linux/Unix systems.

## Features

- Displays current date and system uptime
- Shows CPU load averages (1, 5, and 15 minutes)
- Reports memory usage in a human-readable format
- Displays disk usage for the root filesystem
- Lists top 5 memory-consuming processes

1. Download or clone this repository.
2. Make the script executable:

   chmod +x system_monitor.sh
